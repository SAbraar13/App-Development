**App Development Internship - Week 1**



**Overview:**



This repository contains the projects completed during Week 1 of the App Development Internship. The objective was to set up the Android development environment, learn Android Studio basics, and develop beginner-level Android applications using Java.



**Technologies Used:**



1. Java

2\. Android Studio

3\. Android SDK

4\. Gradle

5\. Git \& GitHub



**Project Structure:**



App-Development/

│

├── HelloWorldApp/

│

├── CounterApp/

│

└── README.md





**Project 1:** HelloWorldApp



**Description:**



A basic Android application developed to verify the Android development environment setup and understand the Android project structure.



**Features:**



1. Displays "Hello World!" on the screen.

2\. Runs successfully on Android Emulator.

3\. Demonstrates Activity and Layout fundamentals.



**Learning Outcomes:**



1. Android Studio setup

2\. Android Emulator configuration

3\. Java integration with Android

4\. Android project structure understanding



**Project 2:** CounterApp



**Description:**



A simple Android application that allows users to increment, decrement, and reset a counter value.



**Features:**



1. Increment counter

2\. Decrement counter

3\. Reset counter

4\. Interactive user interface



**Learning Outcomes:**



1. Button Click Events

2\. TextView Manipulation

3\. Java Event Handling

4\. Android UI Components



**Internship Deliverables Completed:**



1. Android Studio Installation

2\. Java Development Environment Setup

3\. Android Emulator Setup

4\. HelloWorldApp Development

5\. CounterApp Development

6\. GitHub Repository Management
